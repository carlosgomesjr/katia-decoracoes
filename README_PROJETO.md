# P005 - Site Katia Decorações

Landing page em React/Vite para repaginar o site da Katia Decorações usando a referência visual da Liora/Aura e os dados extraídos do site atual da cliente.

## Escopo desta versão

- Página única com navegação por âncoras.
- Hero com imagem da cliente e paleta verde/sálvia/off-white.
- Dobra de produtos com seleção interna e carrossel de fotos, sem abrir outra página.
- Depoimentos de clientes.
- Seção com 3 vídeos selecionados do Instagram por embed oficial.
- Links para WhatsApp, Instagram e Facebook.
- Bloco final com as 3 lojas e telefones.

## Comandos

```powershell
npm run site:dev:katia -- --host 127.0.0.1 --port 5176
npm run site:build:katia
npm run site:lint:katia
```
